# Full stack Chat App with React and Firebase
- live site : https://all-type-chats.netlify.app/

# Tech stacks used ⚙
- ReactJs
- Firebase
- TailwindCSS
- Firebase Storage

# Features 🎇
- chat text message 💬 
- send audio files 🔉
- send video files 🎥
- send images 🖼
- realtime chat 
- Sign in With Google 🔐
- logout
